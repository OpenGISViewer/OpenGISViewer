# OpenGISViewer

A tool designed to view GIS data, with ease of use/installation in mind.

## Benefits

* Cross Platform
* Mobile Deployment





## Technical Details
* Must use Java 17, and have javafx bundled with it. We recommend 17.0.9.fx-zulu
